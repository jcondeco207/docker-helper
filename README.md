# docker-helper

**⚠️ This project is still in development and testing, some features might not work properly ⚠️**

## Purpose

Custom CLI tool made with the purpose of managing my docker containers via terminal.

## Usage

### Option 1

```shell
go run ./cli
```

### Option 2

```shell
cd cli
```

```shell
go build
```

```shell
./cli
```
